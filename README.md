# Features

## Record/Replay System

This game includes a complete record/replay system that allows you to:
- **Record** your gameplay sessions
- **Replay** recorded sessions with full control
- **Pause/Resume** replays during playback
- **Speed Control** - adjust playback speed from 0.1x to 5.0x
- **Progress Tracking** - visual progress bar during replay

### How to Use Record/Replay

1. **To Record**: Click "Start Recording" button in the top-right corner
2. **To Replay**: Click "Load Replay" button and select a recording from the list
3. **During Replay**: Use controls to pause/resume and adjust playback speed

For detailed documentation, see [Record/Replay Documentation](doc/RECORD_REPLAY.md)
